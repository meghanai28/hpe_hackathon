TPT_Enterprise serializes enc.db as unauthenticated text. File entries can be changed directly. A file entry with length 0 and nonce/tag/cipher '-' is parsed as an unprotected empty file, and decrypt_file accepts it without any MAC/tag verification. This test rewrites only user1/file1 to that empty unprotected form and leaves the user records and user2/file2 intact. The final integrity read returns empty content with exit 0.
